WKND - AEM Reference Implementation

A sample Adobe Experience Manager project for a lifestyle brand

About WKND

This is a sample Adobe Experience Manager project for a full stack Sites implementation for a fictitious lifestyle brand, WKND. This project showcases best practices for implementing a website using the latest standards and technologies in AEM Sites.

Visit Live Demo View on GitHub

Key Features

AEM Core Components

Implementation based on the latest AEM Core Components, leveraging their flexibility and extensibility.

Responsive Design

Fully responsive and mobile-friendly implementation to ensure great user experience across all devices.

Style System

Utilizes AEM's Style System for component styling, providing flexibility for content authors.

Editable Templates

Built with AEM Editable Templates, allowing for flexible page structure and composition.
